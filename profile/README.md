## Sensagro

**Satellite water monitoring for farms** — backend API, mobile app, public landing site, and admin portal, wired together with Firebase Auth and shared infrastructure.

### Repositories

| Repository | Role |
|------------|------|
| [**backend**](https://github.com/sensagro/backend) | NestJS API, Prisma, Myriota webhooks, Docker + Azure deploy |
| [**mobile**](https://github.com/sensagro/mobile) | Expo app for farmers |
| [**landing-page**](https://github.com/sensagro/landing-page) | Marketing site (Vite + React, Vercel) |
| [**admin-portal**](https://github.com/sensagro/admin-portal) | Internal admin UI (Vite + React, Vercel) |
| [**documentation**](https://github.com/sensagro/documentation) | Cross-repo docs (deployment, architecture) |

### Documentation

**[Deployment & how everything connects](https://github.com/sensagro/documentation/blob/main/deployment.md)** — Azure backend, GHCR images, Vercel frontends, Firebase, env vars, and local development.
